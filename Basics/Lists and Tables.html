Lists in HTML

There are two primary types of lists in HTML. The unordered list(<ul>), which is used for collections where sequence does not matter, and ordered list (<ol>) where sequence does matter.

Both of these lists use the element list item (<li>) for each item on the list.

Tables

Tables allow you to organize datas in rows & columns. Semantically, "header" tows should be included in the <tbody> element.

Notice the <th> elements automatically bold the content. These hsould be used for header cells. The <caption> element in a table which is displayed by default. Each row is detoned by the <tr> element, with <gt> elements reading within each row.

One of the harder aspects to the master of HTML Tables is the rowspan, which causes the cell to span vertically over rows, and the colspan, which causes the cell to span horizontally over columns.
